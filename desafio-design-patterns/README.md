# Desafio Design Patterns - DIO

Projeto desenvolvido para praticar **Padrões de Projeto (Design Patterns)** em Java.

## Padrões implementados

- Singleton
- Strategy
- Facade

## Objetivo

Demonstrar na prática como os padrões de projeto ajudam a organizar melhor o código e facilitar a manutenção de sistemas.

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos
